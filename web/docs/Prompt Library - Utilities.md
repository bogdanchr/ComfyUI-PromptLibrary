# Prompt Library / Utilities

## Reload Library
Po zapisaniu zmian w plikach TXT:

1. wybierz `reload_library`,
2. ustaw `execute = true`,
3. uruchom kolejkę jeden raz,
4. ustaw `execute = false`,
5. uruchom normalne generowanie.

Operacja czyści cache tekstów i unieważnia poprzednie wyniki node'a Prompt Library. Restart ComfyUI nie jest potrzebny przy dodawaniu lub zmianie wpisów w istniejących plikach.

## Reset Unique
`reset_unique_history` usuwa historię trybu Unique dla jednej kategorii albo całej biblioteki.
